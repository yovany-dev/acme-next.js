## Acme Next.js

Sistema web de facturas del tutorial práctico de la documentación de Next.js con autenticación  y base de datos.

### Credenciales para acceder al sistema
```
user@nextmail.com
```

### Stack
- Next.js
- TypeScript
- TailwindCSS
- PostgresSQL
- NextAuth.js
- Zod
